All HTML headings, <h1> through <h6>, are available by adding `#` code.  |  
All you need to do is to place `_` around the text you want to decorate. |  
With the attractive plugins, you can use Syntax Highlight and Mathjax.   |  


***

>>> `<h1>MDMD</h1>`

>>> `<h4>MDMD</h4>`  
>>> `<h5>MDMD</h5>`  
>>> `<h6>MDMD</h6>`
